# Bee My Lovely GF

An interactive Three.js love letter and walkable detailed pink apartment of memories for Bee.

The site opens inside a soft virtual apartment with a sitting room, desk and laptop corner, music setup, bedroom, kitchen, and balcony. Bee now has a stylized 3D character who can walk around the room with desktop keys or the mobile thumb control. Photo memories are placed room by room, with realistic home details, paper notes, longer letters, Tamambo, Pentagon 5, saved-message panels, Summer Love, and a seamless 270 Degrees Restaurant mural blended into the background wall.

## Local Preview

Serve the folder with any static server, then open the printed URL.

```powershell
python -m http.server 4173
```

The audio starts after pressing `Begin`, which keeps browser autoplay rules happy.
