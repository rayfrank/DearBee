# Bee My Lovely GF

An interactive Three.js love letter for Bee.

The site opens on a rooftop sunset inspired by 270 Degrees in Nairobi, then lets Bee orbit through photo memories, the Pentagon 5 moment, chat-inspired notes, Summer Love, and the final message.

## Local Preview

Serve the folder with any static server, then open the printed URL.

```powershell
python -m http.server 4173
```

The audio starts after pressing `Begin`, which keeps browser autoplay rules happy.
